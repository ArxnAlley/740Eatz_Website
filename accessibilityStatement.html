Accessibility Statement

740Eatz is committed to making this website accessible to all users.

Last Updated: June 30, 2026

Our Commitment

740Eatz is committed to ensuring that our website is accessible to everyone, including people with disabilities. We aim to meet or exceed the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards.

We believe that every visitor should be able to browse our menu, submit an order request, and contact us without barriers. Accessibility is an ongoing effort, and we continually work to improve the experience for all users.

Keyboard Navigation

This website is designed to be fully navigable using a keyboard. Key features include:

  • All interactive elements — links, buttons, and form fields — are reachable via the Tab key.
  • The navigation menu can be opened and closed using keyboard controls.
  • Form submissions and accordions can be activated using the Enter or Space key.
  • Focus states are visible on interactive elements to assist keyboard users.

Screen Readers

This website includes features to support screen reader compatibility:

  • All images include descriptive alt text or are marked as decorative where appropriate.
  • ARIA labels and roles are used for navigation, menus, forms, and interactive components.
  • Page headings follow a logical hierarchy to help screen reader users navigate content.
  • Form fields include associated labels and descriptive placeholder text.

Color and Contrast

We have designed this website with sufficient color contrast between text and background to support users with low vision or color vision differences. Specifically:

  • Primary text uses high-contrast colors against the dark background.
  • Interactive elements such as buttons and links use distinct visual treatments beyond color alone.
  • Status indicators include both color and text labels so information is not conveyed by color alone.

Contact for Assistance

If you experience any difficulty accessing content on this website, or if you need assistance placing an order request, please contact us directly. We are happy to help.

We will respond to accessibility feedback as promptly as possible and work to resolve reported issues.

Future Improvements

Accessibility is an ongoing priority. We continue to evaluate and improve this website through:

  • Regular reviews of page structure, markup, and contrast ratios.
  • Testing with keyboard navigation and screen reader tools.
  • Incorporating feedback from users who encounter barriers.
  • Keeping up with evolving accessibility standards and best practices.

While we strive to meet WCAG 2.1 AA, some third-party content or embedded tools may have limitations outside our direct control. We will note these limitations and work toward solutions as they are identified.
